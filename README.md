# 图神经网络学习

本仓库记录图神经网络（Graph Neural Networks, GNN）的学习过程，涵盖图论基础、数学理论、经典 GNN 模型以及几何深度学习相关内容。

试通过理论学习、代码实现、实验记录和论文阅读，逐步建立对图表示学习和图神经网络的系统理解。
